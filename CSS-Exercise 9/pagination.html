分頁 (Pagination)

用於指向一系列跨頁面相關內容的分頁之文件和範例。

概觀

我們在分頁中使用大量的連結，使它們不易遺漏、能輕鬆擴展，同時提供更大的點擊區域。分頁是使用 HTML 中的列表元素，因此螢幕閱讀器可以讀取可用的連結數量。在外層使用 <nav> 元素讓螢幕閱讀器和其他輔助技術用來識別為導航區塊。

此外，網頁中可能會有更多這樣的導航區塊，因此 <nav> 最好提供具描述性的 aria-label 來表示它的目的性。舉例來說，如果分頁元件被用來搜尋結果區塊，則適當的標籤為 aria-label="Search results pages".

禁用和啟用狀態

分頁可針對不同的情況進行客製。使用 .disabled 來顯示禁用狀態、 .active 顯示當前頁數。

雖然 .disabled 這個 class 使用了 pointer-events: none 來 嘗試 禁用 a 連結功能，這個屬性在 CSS 還是未標準化的，因為鍵盤導覽仍不受影響。因此，您應該在禁用的連結增加 tabindex="-1" ，並使用自定義的 JavaScript 來完全的禁用它。
