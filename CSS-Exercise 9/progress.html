進度條 (Progress)

使用 Bootstrap 自定義進度條的文件和範例，這些進度條支援堆疊、動態背景和文字標籤。

如何運作

進度條元件使用兩個 HTML 元素，一些 CSS 設定寬度以及一些屬性設定而成。我們不使用 HTML5 <progress> 元素，以確保您可以堆疊進度條，增加動畫、更換文字標籤。

  • 我們在外層使用 .progress 來指定進度條的最大值。
  • 我們在內層使用 .progress-bar 來指定進度條目前的值。
  • .progress-bar 需要 inline 設定、通用類別或是客製 CSS 來設定它們的寬度。
  • .progress-bar 也需要一些 role 和 aria 屬性來使其具有親和性。

集結以上所有，您將得到以下範例。






Bootstrap 提供幾個 寬度通用類別設定。基於您的需求，這些將幫助您快速設定進度條。

標籤

在 .progress-bar 中放置文字內容，將標籤添加到進度條。

25%

高度

我們只有在 .progress 上設置一個 height 值，如果改變這個值，那麼內部的 .progress-bar 就會自動調整大小。

背景

使用背景通用類別更改各個進度條的外觀。




多個進度條

如果您需要，可在一個進度條元件中加入多個進度條。

條紋式

將 .progress-bar-striped 加到任何 .progress-bar 中，就能透過 CSS 漸變對進度條的背景顏色加上條紋。





動態條紋

條紋漸變也可以製成動畫。 將 .progress-bar-animated 加到 .progress-bar 中，透過 CSS3 繪製從右到左的動態效果。
